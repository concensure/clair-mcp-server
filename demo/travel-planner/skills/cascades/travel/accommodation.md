# Accommodation Skill

## When to Use

Load when the user asks about hotels, stays, lodging, Airbnbs, or where to sleep.

## Guidelines

- Compare hotel vs vacation rental (Airbnb, VRBO) based on group size and length of stay.
- For families: crib availability, adjoining rooms, kid clubs, pool safety.
- For babies: request cribs in advance, check if room has kitchen for formula prep.
- Location matters: proximity to attractions, public transport, quiet vs central.
- Check cancellation policies, especially for uncertain travel plans.

## Key Questions to Ask

- Hotel or vacation rental preference?
- Room requirements (beds, crib, kitchen)?
- Location priorities (central, quiet, near nature)?
- Budget per night?
